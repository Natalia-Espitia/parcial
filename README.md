## Proyecto: Parcial 
Natalia Espitia Espinel

### Descripción
Este repositorio contiene el proyecto desarrollado como parte del parcial. Aquí se incluyen los archivos y recursos necesarios para ejecutar y entender el proyecto.

### Contenido del Repositorio
- **Código fuente**: Archivos principales del proyecto.
- **Documentación**: Archivos que explican el funcionamiento y uso del proyecto.
- **Recursos adicionales**: Cualquier otro archivo necesario para el proyecto.

### Instalación
1. Clona este repositorio:
    ```bash
    git clone https://github.com/Natalia-Espitia/parcial.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd parcial
    ```
3. Instala las dependencias necesarias:
    ```bash
    mvn package
    ```

### Uso
1. Ejecuta el proyecto:
    ```bash
    mvn spring-boot:run
    ```
